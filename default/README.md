# Login Automation Solution

## Overview
Automated login validation using Python, pytest, and requests. Integrates with DevOps pipelines and follows industry standards for security, maintainability, and scalability.

## Features
- Parameterized credentials via environment variables
- Robust error handling & logging
- Pytest-based test automation
- Ready for CI/CD integration

## Setup Instructions
1. Clone this repo:
   ```bash
   git clone https://github.com/lakshmi1212/AgenticAI.git
   cd AgenticAI
   ```
2. Create and configure environment variables:
   - Copy `.env.template` to `.env` and fill in actual values for `LOGIN_URL`, `LOGIN_EMAIL`, `LOGIN_PASSWORD`.
   - Export variables in your shell or use a tool like `python-dotenv`.
3. Install Python 3.11 and dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running Tests
```bash
pytest Tests/login_test.py --junitxml=Tests/login_test_results.xml
```

## Configuration Steps
- Update `.env` with your endpoint and credentials.
- For CI/CD, ensure secrets are securely managed.

## Maintenance
- Update scripts in `Tests/` for new test cases.
- Update `.env.template` if config changes.
- Review and extend documentation as needed.

## Troubleshooting
- Ensure environment variables are set correctly.
- Check network connectivity to the login endpoint.
- Review pytest logs for details.

## Repository Structure
- `Tests/login_test.py` - Main login test
- `.env.template` - Env var template
- `requirements.txt` - Dependencies
- `README.md` - Documentation
- `.github/workflows/ci.yml` - CI workflow (generated by automation)
- `login_test.metadata.json` - Metadata for pipeline integration
